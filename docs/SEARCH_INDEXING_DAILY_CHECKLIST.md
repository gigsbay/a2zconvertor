# Search Indexing Daily Checklist

Use this during the first 14 days after launch or after routing/SEO changes.

## Daily Checks

- Check sitemap status in Google Search Console.
- Check Page Indexing for new errors.
- Inspect 5 priority URLs.
- Review `Discovered - currently not indexed` and `Crawled - currently not indexed` examples.
- Check 404 errors and fix internal broken links first.
- Check pages with impressions but no clicks.

## Priority URLs

- `/`
- `/launch`
- `/tools`
- `/ai-tools`
- `/pdf-tools`
- `/image-tools`
- `/resources`
- `/convert/png-to-jpg`
- `/convert/jpg-to-png`
- `/convert/hashtag-generator`
- `/convert/instagram-caption-generator`
- `/convert/ai-hook-generator`
- `/convert/ai-video-script-generator`
- `/resources/free-ai-social-media-tools`
- `/resources/best-free-pdf-tools`

## Rules

- Do not request indexing if production has Cloudflare Error 1102, 404s or server errors.
- Do not repeatedly request the same URL every day.
- Indexing can take days or weeks, especially for new domains and new pages.
- If crawled but not indexed, improve content usefulness and internal links before requesting again.
- If discovered but not indexed, add stronger internal links from `/launch`, `/ai-tools`, `/tools` or resource pages.

## What To Improve First

- Title and meta description if impressions appear but CTR is low.
- First screen CTA if clicks land but engagement is weak.
- Sample use cases if users do not understand when to use a tool.
- Related links if users have no obvious next step.
