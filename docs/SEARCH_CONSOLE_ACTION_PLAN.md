# Search Console Action Plan

## Submit The Sitemap

1. Open Google Search Console.
2. Select the `https://a2zconvertor.co.uk/` property.
3. Go to **Sitemaps**.
4. Submit `https://a2zconvertor.co.uk/sitemap.xml`.
5. Confirm the status changes to successful after Google processes it.

## Use URL Inspection

1. Paste a priority URL into the top inspection bar.
2. Confirm Google can access the live page.
3. Use **Test Live URL** after major changes deploy.
4. Use **Request Indexing** for launch-critical pages.

## Daily Checks For The First 14 Days

- Check **Pages** for indexing changes, crawl errors and excluded URLs.
- Check **Sitemaps** for discovered URL count.
- Inspect 3-5 priority URLs manually each day.
- Watch **Performance** for early impressions on AI tool and PDF tool queries.
- Record pages with impressions but low CTR in a simple tracking sheet.
- Check whether newly submitted resource pages are discovered within the first week.
- Confirm no launch pages accidentally show `noindex`.

## If Google Shows "Discovered - Currently Not Indexed"

- Make sure the URL is in the sitemap.
- Add or strengthen internal links from `/`, `/tools`, `/resources`, category pages or related resource pages.
- Improve the page intro and above-the-fold clarity if it feels thin.
- Request indexing only for the most important pages, not every URL repeatedly.
- Wait several days before making more changes unless the page is clearly broken.

## If Google Shows "Crawled - Currently Not Indexed"

- Review whether the page has unique value compared with similar pages.
- Improve title, meta description, intro, FAQ specificity and internal links.
- Add examples, use cases or comparison tables to resource pages where helpful.
- Avoid creating near-duplicate pages that differ only by keyword order.
- Re-request indexing after meaningful content improvements deploy.

## Improve Pages With Impressions But Low Clicks

- Sort Performance by impressions.
- Find pages with low CTR and non-zero impressions.
- Compare the query intent with the title and meta description.
- Rewrite titles to be clearer, not louder.
- Add page-specific descriptions that mention the exact tool or workflow.
- Improve the first heading and intro so search snippets have better context.
- Check whether another A2ZConvertor URL is cannibalizing the same query.

## Monitor Affiliate Landing Pages And AI Tool Pages

- Track impressions and CTR for `/ai-tools`, AI resource pages and AI converter pages weekly.
- Track impressions and CTR for `/pdf-tools`, PDF resource pages and high-value PDF tools weekly.
- Watch affiliate placement pages for high impressions but low clicks; improve helpful content before adding more placements.
- Do not add fake reviews, fake ratings or exaggerated affiliate claims for CTR.
- Compare Search Console query data with CJ, newsletter and Buy Me a Coffee click data.
