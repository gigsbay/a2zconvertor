## Warnings

- If the 404 page appears in top GA4 pages, fix the broken links immediately.
- If dead clicks rise above 5%, review Clarity recordings for confusing UI.
- If bounce is high on tool pages, add clearer CTA copy and example input.
- If impressions grow but CTR is low, rewrite title/meta.
- If pages are crawled but not indexed, improve content and internal links.
